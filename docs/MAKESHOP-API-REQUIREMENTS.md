## 2. 필요한 API 엔드포인트

### 2.1 상품 목록 조회 (Product List)
- **엔드포인트**: `GET /api/v1/products`
- **목적**: 견적서 작성 시 상품 검색/선택
- **필요 파라미터**:
  - `category`: 카테고리 필터링
  - `search`: 상품명 검색
  - `limit`: 페이지당 개수 (기본 100개)
  - `offset`: 페이지네이션
- **필요 응답 필드**:
  - `productId`: 상품 고유 ID
  - `name`: 상품명
  - `price`: 정상가
  - `salePrice`: 판매가 (할인가)
  - `imageUrl`: 대표 이미지 URL
  - `category`: 카테고리명
  - `stock`: 재고 수량
  - `description`: 상품 설명 (선택)

### 2.2 상품 상세 조회 (Product Detail)
- **엔드포인트**: `GET /api/v1/products/{productId}`
- **목적**: 상품 상세 정보 조회
- **필요 응답 필드**: 상품 목록 조회와 동일 + 추가 옵션 정보

### 2.3 카테고리 목록 조회 (Category List) — 선택사항
- **엔드포인트**: `GET /api/v1/categories`
- **목적**: 카테고리별 상품 필터링
- **필요 응답 필드**:
  - `categoryId`: 카테고리 ID
  - `name`: 카테고리명
  - `parentId`: 상위 카테고리 ID
